# АМК-Енисей | Коммерческая недвижимость

Одностраничный веб-сайт (лендинг) для сдачи коммерческой недвижимости компанией **ООО "АМК-Енисей"** в городах Красноярского края.

## 📋 Описание

Современный сайт презентирует три торговых комплекса компании:

- **Торговый комплекс Солонцы** (33 000 кв.м, 2 этажа)
- **Торговый комплекс Ачинск** (14 500 кв.м, 5 этажей)
- **Торговый комплекс Абакан** (4 500 кв.м, 3 этажа)

## 🎨 Дизайн

- **Тёмная тема** с современным минималистичным стилем
- **Адаптивный дизайн** для всех устройств (мобильные, планшеты, десктопы)
- **Плавные анимации** и переходы
- **Интерактивные элементы** с эффектами наведения

## 🔧 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/gubarandrej-bit/amk-enisey-landing.git
cd amk-enisey-landing
```

2. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
python -m http.server 8000
# или
npx http-server
```

3. Откройте браузер и перейдите на `http://localhost:8000`

## 📸 Загрузка изображений

Загрузите ваши изображения в папку `images/`:

```
images/
├── logo.png (логотип компании)
├── solontsy.jpg (фото комплекса)
├── achinsk.jpg (фото комплекса)
└── abakan.jpg (фото комплекса)
```

### Рекомендуемые размеры:
- **Логотип**: 200x200px или больше (квадратный формат)
- **Фото комплексов**: минимум 800x600px (рекомендуется 1200x800px)

## 🌐 Публикация на GitHub Pages

1. Перейдите в Settings репозитория
2. Найдите раздел "Pages"
3. Выберите "Deploy from a branch"
4. Выберите ветку `main` и папку `/root`
5. Сохраните и дождитесь публикации

Ваш сайт будет доступен по адресу: `https://gubarandrej-bit.github.io/amk-enisey-landing/`
